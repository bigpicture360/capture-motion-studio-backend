# Capture Motion Studio Backend

Backend services and worker code for Capture Motion Studio rendering workflows.

## Repository Layout

- `src/` - application and service source code
- `tests/` - automated tests
- `scripts/` - local utility and deployment scripts
- `config/` - configuration examples and deployment settings
- `docs/` - architecture notes, runbooks, and operational docs
- `.github/` - GitHub Actions, issue templates, and pull request template

## Getting Started

Add the backend runtime instructions here once the first service or worker is committed.

## Development

1. Create a local environment file from `.env.example`.
2. Install project dependencies.
3. Run tests before opening a pull request.
